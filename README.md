# PyDetect

Collection of scripts to create detections in various security platforms.

Currently supports:
- Microsoft Sentinel

Planned to support:
- Splunk
- Datadog
- IBM QRadar

## Installation

```
git clone https://github.com/AlbinoGazelle/PyDetect
```

## Usage

```
python3 <tool.py> <required_parameters>
```

## Help

```
python3 <tool.py> -h
```
